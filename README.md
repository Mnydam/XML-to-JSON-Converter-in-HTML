# XML-to-JSON-Converter-in-HTML

AIRAYA XML to JSON Converter with:

Auto-conversion & validation
Cumulative change highlighting (green for JSON)
Error detection with jump-to-line
Multi-file batch processing with detailed error reports
Integrated JSON editor with 2-way sync
Light/Dark themes
Comprehensive help documentation

https://mnydam.github.io/XML-to-JSON-Converter-in-HTML//xml-to-json-converter.html


Download all files and add them to the same directory on your webserver.  Open xml-to-json-converter.html page to run.  Simple.  After conversion and any changes, download to your download directory and you can use the new JSON file.

Tested in IIS.  Start by opening xml-to-json-converter.html
