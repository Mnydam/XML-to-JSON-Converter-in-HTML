üîç Troubleshooting & Error Messages

Common Error Messages

‚ö†Ô∏è "Invalid XML syntax"

What it means: Your XML file has formatting errors.
Common causes:
  • Missing closing tag (e.g., <item> without </item>)
  • Mismatched tags (e.g., <item> closed with </items>)
  • Special characters not escaped (use &lt; for <, &amp; for &)
  • Unclosed attributes (e.g., <item name="test> missing closing quote)
How to fix:
  1. Open your XML file in a text editor
  2. Look for the errors mentioned above
  3. Use an XML validator online to find specific problems
  4. Fix the errors and reload the file
‚ö†Ô∏è "JSON formatting error"

What it means: The conversion process created invalid JSON (rare).
How to fix:
  1. Try the üîÑ Re-Convert button

üîß Technical Details

Conversion Rules

Understanding how XML converts to JSON:

XML Attributes

<person name="John" age="30"> <city>New York</city> </person>

Converts to:

{ "person": { "@attributes": { "name": "John", "age": "30" }, "city": "New York" } }

Repeated Elements (Arrays)

<people> <person>John</person> <person>Jane</person> </people>

Converts to:

{ "people": { "person": ["John", "Jane"] } }

Text Content

<message>Hello World</message>

Converts to:

{ "message": "Hello World" }

‚ùì Frequently Asked Questions

Q: Do I need to install anything?

A: No! This is a web-based tool that runs entirely in your browser. Just open the HTML file.

Q: Is my data sent to a server?

A: No. All conversion happens locally in your browser. Your files never leave your computer.

Q: Can I convert JSON back to XML?

A: This tool converts XML ‚Üí JSON only. For JSON ‚Üí XML, you'd need a different tool.

Q: Why doesn't the "Convert" button do anything?

A: Conversion happens automatically! The button is labeled "Re-Convert" and is only needed if you want to force a refresh.

Q: Can I use this offline?

A: Yes! Download the HTML file and open it locally. It works without internet (except for the JSZip library for batch downloads).

Q: What if my XML has special characters?

A: The converter handles UTF-8 encoding. Special characters should work fine, but ensure they're properly escaped in your XML.

Q: Can I edit the JSON directly?

A: The JSON panel is read-only to prevent errors. Use "Edit in JSON Editor" for a full editing interface.

Q: Why are some lines highlighted in green?

A: Green highlighting shows which lines changed from the original version. The highlights are cumulative - ALL changes you make stay highlighted until you download the file or click "Clear Highlights."

Q: How do I remove the green highlights?

A: Three ways: (1) Download the file - auto-clears highlights, (2) Click the "Clear Highlights" button in the green banner, or (3) Close and reload to start fresh.

Q: Why does it auto-scroll when I edit?

A: After each edit, the JSON output automatically scrolls to center your most recent change. This helps you quickly see what your edit affected without manually searching.
